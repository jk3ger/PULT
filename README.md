# PULT
Приложение для дистанционного управления гуманоидными роботами фирмы STARKIT

Запуск и разрешения 
Приложение работает только в горизонтальной ориентации (landscape). При первом запуске оно запрашивает разрешения на использование Bluetooth и Местоположения (это необходимо для сканирования устройств). Также нужно, чтобы Bluetooth и GPS были включены на самом телефоне. Поиск робота (Scan) На главном экране нажмите кнопку "Scan". Приложение начнет поиск устройств в течение 10 секунд. Оно фильтрует список и показывает только тех роботов, в названии которых есть тег ROKI (например, ROKI-0117). Найденные роботы появятся в выпадающем списке (Spinner).

Подключение 
Выберите нужного робота из списка. Приложение попытается установить соединение через протокол Bluetooth SPP (Classic). Статус подключения отображается в текстовом поле (например, "Connecting...", "Connected to..."). Если связь оборвется, приложение автоматически попытается переподключиться к последнему роботу.

Управление (Джойстики) 
После успешного подключения активируются два виртуальных джойстика (левый и правый). До подключения они полупрозрачные и неактивны. Левый джойстик отвечает за одни оси координат (X, Y), правый — за другие. Данные считываются в реальном времени и преобразуются в байты.

Передача данных
Приложение автоматически отправляет данные роботу каждые 20 миллисекунд (через метод sendJoystickData). Пакет данных формируется с помощью класса CsMessageOut и включает: Заголовок запроса 'R'.◦Координаты X и Y для левого джойстика. Координаты X и Y для правого джойстика.

Отключение
Чтобы разорвать связь, можно нажать кнопку "Disconnect", которая появляется на месте кнопки сканирования после установки соединения.
